# Zorvyn Finance Dashboard

A premium, high-fidelity finance management application built with React, Vite, and Tailwind CSS. Zorvyn provides real-time insights into your spending patterns, transaction trends, and overall financial health.

## Key Features

- **Dashboard Overview**: Comprehensive view of your balance, income, and expenses.
- **Transaction Tracking**: Detailed log of all historical data with interactive filters.
- **Financial Insights**: AI-powered analytics to help you understand your spending habits.
- **Modern UI**: Dark-mode support, glassmorphic elements, and smooth micro-animations.

## Tech Stack

- **Frontend**: React 19, Vite 8, Tailwind CSS 3
- **State Management**: Zustand 5
- **Routing**: React Router 7
- **Charts**: Recharts 3
- **Icons**: Lucide React

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run developement server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
